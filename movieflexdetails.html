Overview

Developed a movie search engine using React and various APIs. The application features movie sliders, movie cards, trailer playback, and language filtering, providing a dynamic and user-friendly interface for exploring and watching films.

Designed and developed an e-commerce application focused on fetching API details, displaying products, and implementing filtering based on price, rating, and categories. The application allows users to navigate to each product to see more images and view a product image gallery.

Key Contributions:

Fetching API Details : Integrated APIs to fetch and display movie data, including images, trailers, and details.

Home Page Design: Created a home page with a carousel featuring movie cards and sections like Top Movies and TV Shows.

Movie Slider: Designed a dedicated slider on the movies page to showcase movies with smooth navigation.

Search Functionality: Implemented a search feature to allow users to find movies easily.

Movie Details Page: Developed a detailed view for each movie, including trailer playback and in-depth information.

Category Sections: Added categories for Popular Movies, TV Shows, and Upcoming Movies, each with similar functionalities and filtering options.

Technologies Utilized:

HTML/CSS: Employed front-end technologies to customize the website's appearance and functionality, ensuring a seamless and visually appealing user experience across all devices.

Bootstrap: Utilized Bootstrap to create a responsive design, ensuring that the application is accessible and visually appealing on a wide range of devices, including desktops, tablets, and smartphones.

React: Implemented various functionalities using React, including fetching and displaying movie details from APIs. Designed a home page with sections for Top Movies and TV Shows, incorporating a carousel for featured content. Utilized React Router for seamless navigation, allowing users to view detailed information and trailers for each movie. Added search functionality and category filters for Popular Movies, TV Shows, and Upcoming Movies, enhancing the interactive and user-friendly experience.

Results: Developed a movie search engine that effectively integrates core features of a modern entertainment platform. Implemented functionalities for detailed movie information, trailer playback, and category-based filtering, providing a rich and user-friendly experience. The application includes a dynamic home page with carousels, search functionality, and well-organized categories such as Popular Movies, TV Shows, and Upcoming Movies. The visually engaging design, intuitive navigation, and optimized performance significantly enhanced user interaction and satisfaction, leading to increased engagement and overall usability.
