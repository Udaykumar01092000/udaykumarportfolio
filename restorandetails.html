Overview

Designed and developed the Swiggy clone application to replicate key functionalities of the original platform, focusing on enhancing user experience and implementing core features.

The application was designed to deliver an engaging and intuitive user experience, with a focus on replicating essential Swiggy functionalities while ensuring smooth navigation and performance. This project demonstrated the effective use of React for building complex, interactive web applications and showcased my ability to integrate third-party APIs to enhance functionality.

Key Contributions:

Calling Swiggy APIs and Displaying Restaurant Details: Implemented the functionality to call Swiggy APIs and display detailed information about various restaurants. This includes displaying the restaurant name, cuisine types, ratings, and other relevant details on a dedicated page.

Search Dishes Functionality: Developed a page that allows users to search for specific dishes by calling the search dishes API. This feature provides real-time search results, enabling users to find their desired dishes quickly and easily.

Search Restaurant Functionality: Created a page that enables users to search for restaurants using the search restaurant API.This functionality returns relevant restaurant search results, making it easier for users to discover new dining options.

Add to Cart Functionality: Integrated an "add to cart" feature using Redux for state management. Users can add dishes to their cart, view their selected items, and manage their orders seamlessly. This feature enhances the overall user experience by providing a smooth and intuitive way to handle food orders.

Search Location Functionality: Developed a feature that allows users to change their location based on their area, enabling them to select nearby restaurants to order from. This was implemented by calling Swiggy's location APIs to provide accurate and relevant restaurant options for users based on their selected location.

Technologies Utilized:

HTML/CSS: Employed front-end technologies to customize the website's appearance and functionality, ensuring a seamless and visually appealing user experience across all devices.

Bootstrap: Utilized Bootstrap to create a responsive design, ensuring that the application is accessible and visually appealing on a wide range of devices, including desktops, tablets, and smartphones.

React: Implemented various functionalities using React, including displaying restaurant details, search dishes functionality, and search restaurant functionality. Leveraged React's component-based architecture to build a dynamic and interactive user interface.

React Redux: Integrated React Redux to manage the state of the application, particularly for the add to cart functionality. This allowed for efficient and scalable state management, enhancing the user experience by providing smooth and intuitive order handling.

Results: Developed a Swiggy app clone that successfully incorporates key features of the original application. The implementation of restaurant details, dish search, and location-based restaurant search functionalities provided a comprehensive and user-friendly experience. The addition of the add to cart functionality using React Redux enhanced the overall user experience by enabling smooth and efficient order management. The visually appealing design, intuitive navigation, and optimized performance contributed to an improved user experience, driving increased engagement and usability of the app.
